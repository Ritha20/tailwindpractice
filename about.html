About

Book Explorer is a lightweight, responsive web app built with HTML, Tailwind CSS, and vanilla JavaScript. Browse and search for books powered by the Google Books API, then save your favorites in your browser using localStorage.

  • Browse results with pagination
  • Full-text search across titles, authors, and topics
  • One-click add/remove favorites
  • Favorites persist in localStorage
  • Responsive layout across mobile, tablet, and desktop
